## О проекте

Сайт-визитка **Данилы Стрельникова** — ведущего мероприятий в Москве (свадьбы, корпоративы, частные события). Стилизован под ретро-приставки NES/Super Mario со Street Fighter элементами, пиксель-артом, CRT-эффектами и игровыми механиками.

**Цель сайта:** максимально продающий, привлекательный и запоминающийся лендинг, который выделяет Данилу среди конкурентов.

**Домен:** danilastrelnikov.ru

## Происхождение и технический долг

Сайт изначально экспортирован из **Tilda**, затем полностью переписан вручную. Старый Tilda-код удалён (files/ папка). Папки css/ и js/ теперь содержат собственный код проекта. Возможные оставшиеся артефакты:

- **`images/`** — файлы именованы в Tilda-формате (`tild<hash>__<name>.<ext>`). По мере работы переименовывать в понятные имена.
- В коде `index.html` могут быть Tilda-атрибуты (`data-tilda-*` и т.п.) — убирать при обнаружении.

## Архитектура

Статический HTML-сайт без сборщиков, фреймворков и пакетных менеджеров.

### Файловая структура

```
index.html          ← Главная (HTML-разметка, ~500 строк)
css/style.css       ← Все стили (~1490 строк)
js/main.js          ← Все скрипты (~200 строк)
policy.html         ← Политика конфиденциальности (/policy), чистый HTML
404.html            ← Страница ошибки в 8-битном стиле
htaccess            ← Apache: HTTPS, /policy → policy.html, 404
robots.txt          ← SEO
sitemap.xml         ← SEO
favicon.ico
images/             ← SVG-декорации, PNG/JPG фото
```

### Структура index.html

Страница организована как игровые "миры" с переходами:

| Section ID | World | Описание |
|---|---|---|
| `#hero` | 1-1 | Герой-секция: ТВ-рамка с видео/фото, стат-бары в стиле Street Fighter |
| `#about` | 1-2 | "Обо мне" — терминал, подземная тема |
| `#achievements` | 1-3 | Карточки ачивок |
| `#files` | 1-4 | Файловый менеджер — папки открывают модалки |
| `#clients` | 2-1 | Список заказчиков — NES-меню выбора |
| `#contact` | FINAL | Контактная форма, телефон, email, реквизиты |
| `#gameover` | — | Game Over с формой розыгрыша |

### Игровые механики (сохранять при рефакторинге!)

- **HUD** — фиксированная шапка: "WORLD X-X", счётчик монет, сердце
- **Сбор монет** — кликабельные SVG-монетки по странице, счётчик в HUD
- **Scroll reveal** — `.reveal` + IntersectionObserver
- **World indicator** — обновление HUD по `[data-world]` секциям
- **Модалки** — `openModal(id)`/`closeModal(id)` для галереи, приколов, цитат, BSOD
- **Галерея** — карусель фото со свайпом и клавиатурной навигацией
- **Загрузочный экран** — фейковый прогресс-бар при загрузке

### CSS — дизайн-система

Все цвета через CSS custom properties в `:root` (палитра NES):
`--sky`, `--brick`, `--pipe-green`, `--coin-gold`, `--fire-red`, `--underground` и т.д.

Шрифт: `Press Start 2P` (Google Fonts). Переменная: `--font`.

## Конвенции

### Стиль кода
- CSS в `css/style.css`, JS в `js/main.js`, HTML-разметка в `index.html`
- CSS: использовать существующие custom properties из `:root`, добавлять новые при необходимости
- JS: vanilla JavaScript, без библиотек. ES5-совместимый стиль (var, function)
- HTML: семантичные секции с `id` и `data-world`

### Дизайн
- Единый 8-битный/пиксельный стиль во всех элементах
- CRT-сканлайн эффект через `body::after`
- Декоративные элементы Mario (трубы, пираньи, монеты, кирпичи) между секциями
- Анимации: пиксельные, ступенчатые (step-end), без плавных градиентов
- Адаптивность: mobile-first, проверять на всех размерах экрана

### SEO
- Язык страницы: русский (`lang="ru"`)
- Мета-теги: title, description, og:title, og:description, og:url, og:type, canonical
- Все ссылки с `https://danilastrelnikov.ru`
- Изображения с осмысленными alt-текстами
- sitemap.xml и robots.txt актуальны

## Разработка

Просто открыть `index.html` в браузере (двойной клик). Никаких серверов и сборки. После правок — обновить страницу (F5).
